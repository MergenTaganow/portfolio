Place your exported resume PDF here:

resume.pdf

Full path:

public/resume.pdf

The header Resume button opens this file in a new browser tab at:

/resume.pdf

Recommended workflow:

1. Open your CV in Word, Pages, or Google Docs.
2. Export it as PDF.
3. Rename it to `resume.pdf`.
4. Put it in this `public` folder.

Opening a PDF in a new tab is the best default for portfolio headers because recruiters can preview it quickly.
If they want to save it, the browser PDF viewer already includes a download button.
